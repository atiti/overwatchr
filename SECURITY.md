# Security Policy

If you find a security issue, please avoid opening a public issue with exploit details.

Instead:

- email the maintainer directly if contact details are available
- or open a private GitHub security advisory if enabled for the repository

Please include:

- a clear description of the issue
- impact
- reproduction steps
- any suggested mitigation

Overwatchr is a local desktop utility, so issues involving Apple Events, Accessibility, shell execution, installer behavior, and hook execution paths are especially relevant.
